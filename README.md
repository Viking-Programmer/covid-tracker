# Covid Tracker using Flask and online API data
* Clone this repository
  ```cmd
  git clone "https://github.com/VikingPathak/covid-tracker.git"
  ```
* Install Flask
  ```cmd
  pip install flask
  ```
* Run the app
  ```cmd
  python covid_tracker.py
  ```
* View the App
  ```cmd
  https://localhost:5000/
  ```
